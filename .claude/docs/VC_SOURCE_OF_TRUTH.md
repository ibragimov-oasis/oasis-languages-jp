# VC_SOURCE_OF_TRUTH

Канонический словарь и базовые константы для документации `vibe-coder`.

## Канонический счётчик репозиториев

- **54 total**
- Формула: **31 original + 23 new**

Исторические числа:
- `31` = первая волна (original cohort)
- `54` = текущая каноническая объединённая система
- Любые другие числа (например `51`) считаются устаревшими для текущей документации, если явно не указано обратное.

## Канонический сценарий поставки

1. Пользователь клонирует `vibe-coder`
2. Переносит **только** `vc-kit/` в свой проект
3. Запускает `bash vc-kit/install.sh` (сначала `--dry-run`)
4. Запускает AI-агента по готовому промту

## Канонические документы запуска

- Entry: `/vibe-code`
- Quickstart: `vc-kit/docs/VC_QUICKSTART.md`
- Full setup RU: `vc-kit/docs/VC_FULL_SETUP_RU.md`
- Prompt pack: `vc-kit/docs/VC_AGENT_PROMPTS_RU.md`
- Interface matrix: `vc-kit/docs/VC_INTERFACE_MATRIX.md`
- Core pipeline: `vc-kit/docs/VC_CORE.md`, `vc-kit/docs/VC_PIPELINE_TRIGGER.md`
